3 How to use latex_curriculum_vitae

3.1 The Main Menue
3.2 Write your application

The Main Menue #

If you already prepared the things in the Installation chapter, we can show how it works in the practice.

  1. Launch Latex Curriculum Vitae from your selected place.

  2. Now opens the user interface:

So you have currently two options. If you want to write your application, you can click on the button and follow the documentation on ???. If you want to see or change your user settings, than you can read more on ???. The database part of this app currently isn't supported yet.

Write your application #

Inside this window you have the possibility to write your job application. Let's look, how it works:

  1. In the jobtitle box, you have to place the title of your new job.

  2. The company name is already known, then you have to type it there.

  3. If you are not knowing the street, zip code and city, you can leave it blank.

  4. Now we can place the name of the contact person. If we dont have any name, then we just can leave it blank. In that case the app will write "Dear Sir or Madam". That also will be used, if you choose the Gender "Unknown". In all other cases it produces a "Dear Mr. Bean" or "Dear Mrs Bean".

    The "Application details" range are pure optional. In the "URL" field you can add the job offer url. It will be placed in the CSV file, what already was discussed.

    If you set the checkbox "Initiative application", the Subject from the letter_of_applicaton.tex will be changed from "Application as" to "Initiative application as".

  5. Now we have two options, to use the app. In case of you are entering a email address, and click on Generate, the app compiles all stuff and send it directly out (if you have set the SMTP settings in the installation part. See ???) with you in BCC. If you leave it blank and click on Generate, the app also compiles all stuff, but in this case it opens the Windows Explorer with the AppData/Local/Temp/latex_curriculum_vitae directory. This is the compile folder. All what you can see there will be deleted each new iteration. But now you have the option to upload the files to a job application website.

  6. Now your application is done. You can now write another one, or you can use the buttons on the bottom.

  7. By clicking on Close will exiting the app. If you are using the button:

    then you forwarded to the online documentation (this one).
